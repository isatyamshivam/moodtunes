## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/moodtunes.git
   cd moodtunes
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to see the app in action!
